# jaekim.ai

Personal portfolio website for Jae Kim, live at [jaekim.ai](https://jaekim.ai).

---

## Features

- **Monolith layout** - full-viewport sections (Hero, About, Experience, Projects, Contact) with mandatory scroll snapping
- **Chrome signature stamp** - liquid-chrome signature rendered behind the hero type and contact heading, revealed with a wipe animation
- **Loader overlay** - signature wipe-in intro that slides away when the page is ready
- **Custom cursor** - dot and trailing ring in difference blend mode, growing over interactive elements (desktop only)
- **Skew-on-scroll** - the whole page subtly skews with scroll velocity, plus hero parallax
- **Signal-in-noise canvas** - the About statement sits on a live canvas waveform that cycles between noise and a clear signal, with a noise-textured shimmer running through the headline text
- **Hover-expand experience rows** - rows invert to cream and expand to reveal role details
- **Inverting project cards** - bordered grid cards that flip to cream on hover with arrow flight
- **Fitted contact heading** - "Let's Talk" outline type scaled by script to span its column
- **Scroll reveals** - staggered rise-in animations driven by IntersectionObserver
- **Reduced-motion support** - animations settle to their final state when the user prefers reduced motion; cursor and hover effects disabled on touch devices

## Tech Stack

- Vite (vanilla HTML/CSS/JS, no framework)
- Cloudflare Pages
- Archivo and Space Grotesk (Google Fonts)

## 주요 기능

- **모놀리스 레이아웃** - 전체 화면 섹션(히어로, 소개, 경력, 프로젝트, 연락처)과 스크롤 스냅
- **크롬 서명 스탬프** - 히어로 타이포와 연락처 헤딩 뒤에 배치된 리퀴드 크롬 서명, 와이프 애니메이션으로 등장
- **로더 오버레이** - 서명이 그려지는 인트로 화면, 준비 완료 시 위로 슬라이드
- **커스텀 커서** - difference 블렌드 모드의 점과 링 커서, 인터랙티브 요소 위에서 확대 (데스크톱 전용)
- **스크롤 스큐** - 스크롤 속도에 따라 페이지 전체가 미세하게 기울어지는 효과와 히어로 패럴랙스
- **시그널 캔버스** - 소개 문구 뒤에서 노이즈와 신호 사이를 순환하는 실시간 파형, 헤드라인 텍스트에는 노이즈 질감 시머 효과
- **호버 확장 경력 행** - 호버 시 크림색으로 반전되며 상세 설명이 펼쳐지는 행
- **반전 프로젝트 카드** - 호버 시 크림색으로 반전되고 화살표가 날아가는 테두리 그리드 카드
- **맞춤 연락처 헤딩** - 컬럼 너비에 맞게 스크립트로 스케일되는 "Let's Talk" 아웃라인 타이포
- **스크롤 리빌** - IntersectionObserver 기반의 순차 등장 애니메이션
- **모션 최소화 지원** - 모션 최소화 설정 시 애니메이션이 최종 상태로 고정; 터치 기기에서는 커서·호버 효과 비활성화

## 기술 스택

- Vite (바닐라 HTML/CSS/JS, 프레임워크 없음)
- Cloudflare Pages
- Archivo, Space Grotesk (Google Fonts)
